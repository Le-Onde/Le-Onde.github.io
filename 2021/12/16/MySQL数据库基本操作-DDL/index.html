MySQL数据库基本操作 - DDL

MySQL数据库基本操作 - DDL

Ⅰ.DDL解释

DDL（Data Definition Language），数据定义语言，该语言包括以下部分

  • 对数据库的操作
  • 对表结构的常用操作
  • 修改表操作

Ⅱ.对数据库的常用操作

功能 SQL
查看所有的数据库 show databases;
创建数据库 create database[if not exists] mybd1 [charset=uft8];
切换数据库 use mydb2;
删除数据库 drop database [if exists] mybd1;
修改数据库编码 alter database mybd2 character set utf8;

Ⅲ.对表结构的常用操作

1）创建表格式

创建表是构建一张空表，指定这个表的名字，这个表有几列，每一列叫什么名字，以及每一列存储的数据类型。

2）数据类型

数据类型是指在创建表的时候为表中字段指定数据类型，只有数据符合类型要求才能存储起来，使用数据类型的原则：够用就行，尽量使用取值范围小的，而不用大的，这样可以更多的节省存储空间
